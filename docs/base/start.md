## 新建项目

游戏模板

- Core - Template：只保留一些必要内容的模板
- ARPG 事件 UI：UI 交互由事件指令实现的 ARPG 模板
- ARPG 脚本 UI：UI 交互由 Javascript 脚本实现的 ARPG 模板
- 最小化当前项目：将当前打开的项目，导出到另一个位置，同时去掉未用到的文件
